Primitives

null

The null value represents “nothing” or “no value”.

var x = null;

boolean

Boolean values are either true or false.

var is_valid = true;
if (!is_valid) {
    // ...
}

number

Blaze numbers support integer and floating-point values:

var a = 42;
var b = 3.14;

Arithmetic operations: +, -, *, /, %, ==, <, >.
